Нельзя просто так взять и запустить таск по-умолчанию.

Основные команды системы:
'gulp build' - сборка кода проекта
'gulp dev' - сборка проекта, отслеживание изменений, поднятие локального сервера, автообновление страницы
'npm run release' - сборка кода проекта готового для продакшен использования

Выполните:
$ gulp --tasks
чтобы узнать о всех тасках системы
